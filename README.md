# Consultant Allocation Management

Consultant Allocation Management is a project to help manage the allocation of consultants to projects.

## What it does

- Keeps track of consultants and their availability
- Manages project assignments / allocations
- Helps visualize who is allocated to what and when

## Goals

Provide a simple way to plan and maintain consultant assignments, avoid scheduling conflicts, and support basic allocation reporting.
